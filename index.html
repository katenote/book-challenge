강력하고 똑똑한 AI 개발 파트너 Antigravity의 세계에 오신 것을 환영합니다. 아래 로드맵을 따라 단 몇 분 만에 첫 코딩 여정을 설계해 보세요.

01. 온보딩 로드맵

1단계: 작업 공간(Workspace) 준비하기

Antigravity는 실제 폴더의 모든 코드를 이해하고 작성하는 강력한 에이전트입니다.

필수
빈 디렉토리(폴더)를 생성하세요.
데스크톱이나 안전한 경로에 개발할 프로젝트 명으로 새로운 폴더를 생성합니다. (예: `my-awesome-app`)
연결
해당 폴더를 VS Code 등 편집기에서 여세요.
Antigravity가 해당 위치를 기준으로 파일을 생성하고 실행 명령어를 제안할 수 있도록 폴더 구조가 지정됩니다.
Terminal
# 1. 새 프로젝트 폴더 만들기 mkdir my-awesome-app cd my-awesome-app # 2. 에디터로 폴더 진입하기 code .

2단계: Antigravity와 인사하기

설치된 시스템에서 Antigravity 에이전트를 불러와 대화를 시작할 수 있습니다.

입력
채팅 창에 가벼운 인사를 건네보세요.
"안녕 Antigravity!" 또는 "여기서 무엇을 할 수 있니?"라고 입력하여 대화창이 올바르게 연결되었는지 테스트합니다.
기반
시스템 상태를 자동으로 확인합니다.
Antigravity는 사용자의 OS 종류(Windows 등)와 디렉토리 권한, 설정된 모델 정보를 로드하여 최적화된 상태로 대기합니다.
Antigravity System
User: 안녕 System: OS=Windows | Workspace=Active Antigravity: 안녕하세요! 무엇을 만들어볼까요?

3단계: 페어 프로그래밍과 파일 생성

단순 답변을 넘어, Antigravity가 직접 개발 환경에 파일을 구성합니다.

요청
구체적인 기능을 한글로 요청해 보세요.
"HTML/CSS로 예쁜 카운터 앱 만들어줘" 같이 만들고자 하는 프로그램의 기능을 자연어로 명확하게 지시합니다.
생성
실시간 파일 작성 및 코드 삽입을 지켜봅니다.
가이드된 코드를 직접 복사/붙여넣기 할 필요 없이, Antigravity가 작업 공간에 완벽한 코드 파일을 조립해 나갑니다.
Action log
⚙️ Tool call: write_to_file 📁 Path: /index.html ✅ Status: Success ⚙️ Tool call: write_to_file 📁 Path: /style.css ✅ Status: Success

4단계: 스마트한 계획 수립과 도구 활용

계획 모드(Planning Mode)를 통해 더 복잡하고 안정성 높은 개발 프로세스를 유지합니다.

계획
큰 규모의 수정에는 계획 모드가 켜집니다.
Antigravity는 복잡한 변경을 시작하기 전, `implementation_plan.md` 파일을 생성하여 사용자 승인을 먼저 받습니다.
자동화
터미널 실행 및 수동 빌드를 위임하세요.
`npm run dev`나 Python 스크립트 실행 등 복잡한 실행 명령어도 Antigravity가 안전하게 구성하여 쉘에서 구동합니다.
Planning Mode

03. 필수 슬래시 명령어 도감

/goal

시간이 오래 걸리거나 깊은 탐색이 필요한 복잡한 목표가 있을 때 사용합니다. 에이전트가 극도로 꼼꼼하게 끝까지 추적합니다.

입력 예시:
"/goal 이 프로젝트에 테스트 코드를 완벽하게 추가하고 커밋해줘"
/schedule

특정 타이머를 설정해두고 작업을 연기하거나, 주기적으로 상태 체크가 필요할 때 백그라운드 작업을 실행합니다.

입력 예시:
"/schedule 1분 후에 현재 빌드가 완료되었는지 검사해줘"
/browser

인터넷 브라우징, 신규 오픈소스 명세 문서 탐색, 리액트 웹 뷰 스크린샷 캡처 및 레이아웃 상태를 평가할 때 사용합니다.

입력 예시:
"/browser 최신 MDN의 CSS grid 속성에 대해 찾아서 요약해줘"
/grill-me

요구 사항이 다소 모호하거나 복잡할 때 대화형 심층 인터뷰를 진행하여 최적의 비즈니스 논리와 테마를 정립합니다.

입력 예시:
"/grill-me 쇼핑몰의 세부 기획과 데이터베이스 구조를 같이 정해줘"
